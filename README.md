# sales_dashboard_swiggy_excel1
An end-to-end data analytics project analyzing Swiggy sales data to identify revenue trends, customer patterns, and peak order timings, supported by interactive dashboards and data-driven insights.
This project focuses on analyzing sales data from Swiggy using Advanced Excel to generate meaningful business insights. An interactive dashboard was developed to track key performance metrics and support data-driven decision-making.
Objectives
Analyze large-scale food delivery sales data
Track revenue and order trends over time
Evaluate customer behavior and ratings
Identify top-performing cities and categories
Total Orders: 197,000+
Total Sales: ₹53M+
Time Period: Nov 2025 – Dec 2025
